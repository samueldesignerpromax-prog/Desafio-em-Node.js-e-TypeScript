# Podcast Manager API

## Descrição

Este projeto consiste em uma API simples para gerenciamento de podcasts desenvolvida utilizando Node.js e TypeScript sem frameworks.

O objetivo é compreender como criar uma API HTTP pura utilizando os recursos nativos do Node.js.

---

## Tecnologias utilizadas

- Node.js
- TypeScript
- HTTP Module

---

## Funcionalidades

- Listar podcasts
- Organizar episódios
- Gerenciar dados de podcasts
- Criar API simples sem frameworks

---

## Instalação

Clone o repositório:

```
git clone https://github.com/seuusuario/podcast-manager-node-ts
```

Entre na pasta:

```
cd podcast-manager-node-ts
```

Instale dependências:

```
npm install
```

---

## Execução

Compilar TypeScript:

```
npx tsc
```

Rodar o servidor:

```
node dist/server.js
```

---

## Teste da API

Endpoint disponível:

```
GET /podcasts
```

Exemplo:

```
http://localhost:3333/podcasts
```
